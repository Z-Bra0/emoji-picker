# Emoji Picker

A simple, ad-free emoji search page. No installs, no tracking, no clutter.

**[Live Demo](https://github.com)**

## Features

- Search emojis by name
- Browse by category (Smileys, People, Animals, Food, Travel, Activities, Objects, Symbols)
- Click any emoji to copy it to clipboard
- Dark mode toggle
- 1000+ emojis
